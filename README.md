# TalesOfShadows
Indie Video Game

All stories are based on peasant revolutions in Hunan province durring the 1926s
Follow the story of 7-11 to
  Interact with NPCs
  Defeat the Baron and his soldiers
  Amazing gameplay
  Learn about the world
  Free innocent peasants from opression
